# Game Tests

Integration and unit tests for engine-level game logic. Uses the [Catch2](https://github.com/catchorg/Catch2) framework.

## Contents
- **CombatTests.cpp** — Weapon attribute formulas and module scaling verification.
- **EconomyVolumeTests.cpp** — Economy volume and trade capacity tests.
- **ShipViabilityTests.cpp** — Blueprint viability and resource provisioning tests.

## Structure
- **1. Objective:** What goal was achieved?
- **2. Changes Made:** High-level list of modified components.
- **3. Evidence:** Screenshots, terminal logs, or recordings demonstrating the result.
- **4. Verification:** Links to passed quality gates or automated tests.

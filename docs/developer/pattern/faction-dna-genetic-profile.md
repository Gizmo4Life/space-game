# Pattern: Faction DNA Genetic Profile

## 1. Geometry
- **Requirement:** Define a multi-axis strategic profile using normalized floats `[0.0, 1.0]`.
- **Requirement:** Core axes must include **Aggression**, **Industrialism**, and **Commercialism**.
- **Rule:** DNA is randomized at faction initialization to ensure diverse galaxy-scale behavior.
- **Rule:** High `Aggression` biases ships towards Combat roles and increased weapon density.
- **Rule:** High `Industrialism` increases factory efficiency and resource extraction priority.
- **Rule:** High `Commercialism` biases ships towards Cargo roles and increases trade frequency.

## 2. Nuance
FactionDNA serves as the "genetic code" for procedural growth. It isn't a static choice but a set of weights that influence every secondary system—from the hulls generated by the `HullGenerator` to the expansion choices made by the `EconomyManager`. This ensures that even two factions with the same name or color will behave differently based on their underlying DNA.

## 3. Verify
- A faction with `Aggression > 0.8` builds more weapons than a faction with `Aggression < 0.2`.
- Procedural hull generation reflects DNA biases (e.g., more hardpoints for aggressive factions).
- Logs show DNA being correctly passed to `HullGenerator` and `EconomyManager`.

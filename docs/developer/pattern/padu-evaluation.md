## Structure
- **Preferred (P):** The strategic, primary target shape.
- **Acceptable (A):** Functional shape supported for legacy or niche use cases.
- **Discouraged (D):** Functional shape carrying technical debt; target for refactoring.
- **Unacceptable (U):** Forbidden shape; blocks CI/CD and PR approval.

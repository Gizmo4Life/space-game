## Structure
- **Requirement:** Explicitly define the primary and secondary on-call ownership for every module.
- **Rule:** Define hard-stop criteria for when an incident must be escalated to leadership or specialized teams.
- **Verify:** Escalation contacts must be confirmed valid and reachable in the [Universal Runbook](/docs/operational/runbook/universal-runbook.md).
